# Aether Neural Lab

A web-based interactive neural network laboratory for exploring, visualizing, and experimenting with neural network concepts.

## 🌟 Features

- **Interactive Neural Network Visualization** - Visualize neural networks in real-time with dynamic node and connection rendering
- **Web-Based Interface** - No installation required; run directly in your browser
- **Responsive Design** - Optimized for desktop and tablet viewing with beautiful CSS styling
- **JavaScript-Powered** - Fast, client-side computation and interaction

## 📊 Project Composition

This project is built with:
- **CSS** - 55.8% (Styling and layout)
- **HTML** - 24.1% (Structure and markup)
- **JavaScript** - 20.1% (Interactivity and logic)

## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/noswad/aether-neural-lab.git
   cd aether-neural-lab
   ```

2. Open the application:
   - Open `index.html` in your web browser
   - Or serve the files using a local server:
     ```bash
     python -m http.server 8000
     ```
   - Then navigate to `http://localhost:8000`

## 🎯 How to Use

1. **Load the Application** - Open index.html in your browser
2. **Explore Neural Networks** - Interact with pre-built network visualizations
3. **Experiment** - Modify network parameters and observe changes in real-time
4. **Learn** - Understand how neural networks process information visually

## 🛠️ Technologies Used

- **HTML5** - Semantic markup for web content structure
- **CSS3** - Modern styling with animations and responsive design
- **JavaScript (ES6+)** - Core computational and interactive logic
- **Canvas API** - For neural network visualization and rendering
